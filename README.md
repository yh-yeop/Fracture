# Metroidvania(제목 미정)
메트로배니아 장르의 게임이며 Unity로 개발 예정



# 기획

## 1. 게임의 기본 틀/시스템
    정해진 맵을 탐험(=오픈월드 X)

    탐험 중 얻을 수단을 사용해야만 지나갈 수 있는 길 존재
        - ex) 1번째 맵에서 발견했지만 못 지나가던 길을 2번째 맵에서 얻은 아이템/스킬을 활용해 지나감
    
    언젠가 이전 지점으로 돌아가서 해결해야할 일이 생김
        - ex) 3스테이지에서 4스테이지로 가던 도중 암호가 필요. 사실 1스테이지에 있던 수상한 요소를 분석해서 얻어내야 했던 것.
