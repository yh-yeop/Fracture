# Metroidvania(제목 미정)
메트로배니아 장르의 게임이며 Unity로 개발 예정



# 기획

## 1. 게임의 기본 틀/시스템
    정해진 맵을 탐험(=오픈월드 X)

    탐험 중 얻을 수단을 사용해야만 지나갈 수 있는 길 존재
        - ex) 1번째 맵에서 발견했지만 못 지나가던 길을 2번째 맵에서 얻은 아이템/스킬을 활용해 지나감
    
    언젠가 이전 지점으로 돌아가서 해결해야할 일이 생김
        - ex) 3스테이지에서 4스테이지로 가던 도중 암호가 필요. 사실 1스테이지에 있던 수상한 요소를 분석해서 얻어내야 했던 것.

## 2. 세부사항
    2D를 전제로 결정

    - 횡스크롤

        - 중력 O
    
    - 배경
        - 강철 재질 연구소
        - 느낌
            - 이질적이게 넓고 깔끔함
            - 연구 실패의 흔적. 고장나고 황폐함
            - 가동중이지만 사람의 흔적은 X




## 자유기획

연구실 내 노트북에 접근 시 시점을 탑뷰로 변환 및 데이터 세계에서 플레이.
횡스크롤때는 유틸적으로 도구를 사용하고 탑뷰 시점일 때 전투 컨텐츠를 집어넣기




## 2026.02.25 기획

## 1. 프로젝트 개요
### 장르
2D 플랫포머 메트로배니아

### 핵심 키워드

- 사이버펑크
- 시간 붕괴
- 시스템 vs 인간
- 불완전함
- 플레이 성향
- 빠르고 민첩한 이동
- 스태미나 기반 전투
- 제한적 시간 능력 활용
- 과도한 설명 없음

### 2. 세계관 배경
- 전쟁 이후
- 잦은 전쟁
- 자원 고갈
- 환경 황폐화
- 인류는 생존을 위해 거대한 기계도시를 건설.
- 기계도시
- 중앙 통제형 사회
- 화이트리스트 기반 존재 관리
- 계층 구조 고착화
- 시간 간섭 기술 개발
- 고위층은 “회귀”를 극구 반대.
- 시간 간섭 장치
- 시간축 관측 및 간섭 가능
- 회귀 가능성 존재
- 실험은 중단 상태
### 3. 주인공

- 전쟁 고아
- 정부가 거둬 실험체로 활용
- 시간 간섭 프로토타입 실험자
- 현재는 개인 레지스탕스 (단독 행동)
- 성격: 사색형, 감정, 표현 최소, 결단형
- 핵심 신념: 불완전함이 인간이다.
### 4. 게임 시작 구조
- 프롤로그 – 연구소 내부
- 상태: 기본 대시, 근접 전투 장비, 숙련된 전투 감각
#### 전개
- 연구소 잠입
- 일반 경비 로봇 전투
- 경보 발령
- “화이트리스트 미등록 개체 발견.”
- “물리적 제압 개시.”
- 장치 작동 시도
- 추격
- 과부하 발생
- 시간축 붕괴
- 암전

### 5. 본편 시작

- 주인공은 외곽 폐허로 튕겨나감.
- 상태: 부상, 능력 일부 약화, 시간 능력 불안정, 이후 능력 점진 회복 및 확장.

### 6. 핵심 시스템
#### 이동
- 대시 기반
- 벽 이동
- 공중 확장 (획득형)
- 빠르고 민첩한 템포 유지.
#### 전투
- 스태미나 소모형
- 일부 능력은 체력 공유
- 패턴 분석형 전투
#### 시간 능력

- 단기 회귀 (제한적 상황 활용)
- 특정 구역 시간 전환
- 시간 잔상 응용
- 프롤로그보다 후반부가 더 강력해짐.

#### 사망 시스템

- 화면 색 빠짐
- 균열 연출
- 암전
- 째깍째깍 소리
- 체크포인트 부활
- 스토리적 설명 없음.
- 메타 시스템으로만 존재.

### 7. 적 구성
#### 프롤로그 & 초반

- 경비 로봇
- 감시 드론
- 진압 병기
- 톤: 시스템적, 감정 없음

#### 중반

- 반기계화 인간 개조체
- 시간 붕괴 잔상체
- 톤: 인간성 왜곡

#### 후반
- 인체형 기계 (미래 인류)
- 집단 동기화 전투
- 감정 완전 제거
- 톤: 완전함의 공포

### 8. 시간 붕괴 컨셉

- 구역 진입 시 시간축 왜곡
- 과거/현재/미래 잔재 공존
- 특정 시간대에서만 활성화되는 발판/적
- 시간은 무기가 아니라 환경이다.

### 9. 보스 구조

- 과부하 진압 로봇 (시스템 오작동)
- 하층 감독 개조체 (인간성 왜곡)
- 시간 고정 장치 (질서 vs 붕괴)
- 동기화 지휘 개체 (개인의 소멸)
- 중앙 관리 코어 (완전함)
- 보스는 철학 단계와 연동됨.

### 10. 엔딩

- 주인공 최고 관리자 권한 획득.
- 권력 선택 가능.
- 하지만 회귀 선택.
- 마지막 대사: “불완전함이 있어야 인간 아니겠어?”

- 암전.
- 완전 무음.
- 되돌림 없음.

### 11. 작품 메시지

- 완전함은 인간성을 제거한다.
- 분쟁은 인간의 일부다.
- 불완전함이 인간을 인간답게 만든다.

### 12. 개발 방향성

- 설명 최소화
- 과도한 컷신 지양
- 이동감 최우선
- 시간 연출은 과장보다 위화감 중심
