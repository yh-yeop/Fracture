# Metroidvania(제목 미정)
메트로배니아 장르의 게임이며 Unity로 개발 예정



# 기획

## 1. 게임의 기본 틀/시스템
    정해진 맵을 탐험(=오픈월드 X)

    탐험 중 얻을 수단을 사용해야만 지나갈 수 있는 길 존재
        - ex) 1번째 맵에서 발견했지만 못 지나가던 길을 2번째 맵에서 얻은 아이템/스킬을 활용해 지나감

    퍼즐 요소와 전투 요소를 섞어 지루하지 않게 만듬
        - 플레이어의 자유도는 어느정도 보장하나 게임의 진행을 위해서는 정답을 찾아야 하는 구조
            - 웬만하면 모든 상황에서 2개 이상의 방법을 제작
